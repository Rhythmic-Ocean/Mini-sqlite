<h1>Mini-sqlite</h1>

<h2>After you compile the .c files as output, run the following command on the terminal</h2>

<h3>./output table1

(this creats/opens table1.csv where you can create ONE table i.e it's only one table per .csv file)</h3>


<h2>The DBSM interface currently only has 3 functions:</h2>

<h3><p>CREATE tablehead tablehead tablehead...... (creates a table head with given table heads)</p>

INSERT tabledata tabledata tabledata......(adds tabledata to to tablehead in order)

READ ALL (displays everything in table)</h3>
